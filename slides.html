Warum R, und nicht SPSS/Mplus oder Python/Julia?

  • R ist eine Statistik-Programmiersprache
  • SPSS ist ein point&click Tool; ja ich weiss, man kann auch Syntax schreiben
  • Mplus benutzt eine eigens entwickelte (beschrnkte) Makrosprache
  • Python ist eine allgemeine Programmiersprache, nicht spezifisch fr Statistik
  • Julia ist eine Programmiersprache entwickelt fr allgemeine wissenschaftliche Zwecke

Warum RStudio, und nicht R-GUI oder Emacs?

  • Das R-GUI ist sehr beschrnkt
  • Emacs/ESS ist sehr unbeschrnkt
  • In RStudio gibt es sehr vieles, das man per Mausklick machen kann. Im Allgemeinen sind das keine statistischen Prozeduren o.., sonder alles Dinge, die das IDE selbst betreffen, wie z.B. Dateien ffnen oder ein Skript kompilieren. So ziemlich alles ist aber auch per Skript mglich.
